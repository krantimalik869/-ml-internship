\# Task 2: Emotion Recognition from Speech



\## Objective

Recognize human emotions (happy, angry, sad, neutral, fearful) 

from speech audio using deep learning.



\## Approach

Deep learning with speech signal processing techniques.



\## Key Features

\- Extract MFCC (Mel-Frequency Cepstral Coefficients) features

\- CNN model for emotion classification

\- Visualize training history and confusion matrix



\## Algorithms Used

\- CNN (Convolutional Neural Network)

\- LSTM (Long Short-Term Memory)



\## Datasets (Real Project)

\- RAVDESS

\- TESS

\- EMO-DB



\## How to Run

pip install -r requirements.txt

python emotion\_recognition.py



\## Results

\- Model trained on 5 emotion classes

\- Metrics: Accuracy, Precision, Recall, F1-Score
