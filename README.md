# \# Task 1: Credit Scoring Model

# 

# \## Objective

# Predict creditworthiness of individuals using past financial data.

# 

# \## Algorithms Used

# \- Logistic Regression

# \- Decision Tree

# \- Random Forest

# 

# \## Metrics

# \- Precision, Recall, F1-Score, ROC-AUC

# 

# \## Dataset

# Synthetic dataset generated using sklearn.

# Features: income, debt\_amount, credit\_history\_years, 

# num\_credit\_cards, payment\_history, age, employment\_years, 

# loan\_amount, monthly\_expenses, num\_late\_payments

# 

# \## How to Run

# pip install -r requirements.txt

# python credit\_scoring.py

# 

# \## Results

# \- Best Model: Random Forest

# \- ROC-AUC: \~0.90
